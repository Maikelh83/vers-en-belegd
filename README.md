# Vers en Belegd — website

Broncode voor www.versenbelegd.nl (zakelijke lunch, Veenendaal e.o.).

## Structuur

- `index.html` — homepage (menu, bestellen, zakelijk, scholen, vergaderlunch)
- `privacy.html` — privacyverklaring
- `images/` — logo, favicons, hero- en menufoto's (compleet, webp + png)

## Lokaal bekijken

Open `index.html` in de browser, geen buildstap nodig.

## Relatie met Boslaantje Doen

Losstaande website/domein, gelinkt vanuit en naar www.boslaantjedoen.nl. Aparte repo, want ander domein en geen gedeelde broncode.
